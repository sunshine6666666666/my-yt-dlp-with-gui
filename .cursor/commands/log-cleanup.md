# Log Cleanup - 日志一键清理

## 使用方法

```bash
/log-cleanup
```

## 上下文

- **核心任务**：无条件删除当前项目的 `debug.log` 和 `app.log`。
- **清理目标**：释放磁盘空间，确保下次调试时日志环境纯净。

## 执行指令

请立即执行以下清理操作：

1. **删除运行时插桩日志**：
   - 路径：`.cursor/debug.log`
2. **删除业务逻辑日志**：
   - 路径：`yt-dlp-gui/logs/app.log`

## 确认与反馈

清理完成后，请简洁回复告知清理结果。
