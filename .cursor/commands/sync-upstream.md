# Upstream Sync - 官方代码同步

## 使用方法

```bash
/sync-upstream
```

## 执行指令

请立即执行以下操作，**无需询问确认**：

1.  **环境校验**：
    - 检查是否存在名为 `upstream` 的远程仓库，若不存在则添加：`https://github.com/yt-dlp/yt-dlp.git`。
    - 确保当前分支为 `main`。

2.  **代码同步**：
    - 执行 `git fetch upstream` 获取官方最新数据。
    - 执行 `git merge upstream/master --no-edit` 将官方更新合并到本地。

3.  **变动审计**：
    - 如果合并成功，提取并分析本次合并引入的核心变动（关注 `yt_dlp/` 核心逻辑、`extractor/` 解析器更新等）。

## 确认与反馈

同步完成后，请使用 **中文** 详细告知我：
- 本次合并了官方的哪些核心改动？
- 是否存在合并冲突（如有，请列出文件路径）？
- 提醒我手动测试 GUI 功能是否正常。

> **注意**：此指令仅执行本地合并，**严禁执行 `git push`**，以便我进行人工测试。
